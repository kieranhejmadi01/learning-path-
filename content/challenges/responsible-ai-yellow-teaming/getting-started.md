## Prepare your approach

These skills and resources are helpful before you begin:

- Proficiency in Python and PyTorch if you deploy a private model
- Familiarity with Hugging Face and LLM prompting techniques
- Access to an Arm-based cloud instance if you host a private model
- Access to a public LLM if you prefer a hosted workflow
